git https 方式 操作:

1. git clone https://github.com/boobusy/boobusy.github.io.git
2. git add * {添加修改的文件到 暂存区 [index] }
3. git commit -m "代码提交信息" { 缓存区 [head]（最后提交的结果）,也就是提交到分支 }
3. git push origin master { 将改动推送到远程主机 }
git remote -v { 查看本地所有主机 }
git remote rename origin git_busy_blog { 给主机改名字 }
git remote set-url git_busy_blog https://您的github用户名@github.com/boobusy/boobusy.github.io.git { 修改url }
git push git_busy_blog master { 提交,会提升输入密码 }
